## [v0.5.1] - 2026-04-27

### ✨ 新功能
- ⌨️ **快捷键支持** — 新增 Ctrl+F（聚焦搜索）、F5（扫描）、Ctrl+A（全选）、Delete（删除）、←/→（预览导航）

### 🔧 性能优化
- **缩略图缓存** — LRU 淘汰改为增量计数（`_current_cache_bytes` 全局追踪），消除每 50 张触发全目录扫描的性能抖动；仅在超过上限时才触发淘汰
- **数据库标签查询** — `get_all_tags()` 改用 SQLite `json_each()` 直接提取标签，Python 端零 JSON 解析

### 🔧 Code Review 修复
- **`_touch_thumb` bug 修复** — 访问已有缩略图时不再重复累加文件大小到内存计数器，避免计数器漂移导致的误淘汰
- **批量标签操作** — `rename_tag()` / `merge_tags()` / `delete_tag()` 改为 `executemany` 批量执行 + 单事务提交，替代逐条更新
- **Schema v3 迁移** — 新增 `idx_content_rating` 复合索引，加速内容分级过滤查询

### 📊 测试
- 216 个测试全绿通过（+11 新增覆盖新增代码路径）

## [v0.5.0] - 2026-04-27

### ✨ 新功能
- 🏷️ **内容分级汉化** — 分级筛选和预览详情中显示中文（所有人/青少年/成人/仅成人）
- 📝 **工坊 ID 本地化** — 预览对话框中 `Workshop ID` → `工坊 ID`

### 🐛 修复
- **运行立即退出** — 修复 `app.py` 缺少 `if __name__ == "__main__": main()` 入口导致程序启动后立即退出的 bug
- **壁纸默认样式** — 壁纸适应模式默认值从 `stretch`（拉伸）改为 `fill`（填充），保持图片比例不变形

### 🔧 Code Review 修复
- **SQL LIKE 转义** — 搜索关键词中的 `%` 和 `_` 通配符现在正确转义，避免意外匹配
- **批量操作优化** — 批量收藏/取消收藏从 N+1 单条 SQL 改为单次 `IN` 查询，大幅提升大量选中时的性能
- **视频播放器资源泄漏** — `QMediaPlayer`/`QAudioOutput` 设置 parent 对象，关闭时正确断开视频输出
- **QTimer parent** — 预览对话框 resize 防抖定时器设置 parent，符合 Qt 对象树最佳实践
- **命名冲突修复** — `FilterBar.RATING_LABELS` 类属性与模块导入同名问题
- **import 位置优化** — `models.py` 属性方法内的 `import os` 移到文件顶部，改用 `pathlib.Path`
- **去重逻辑简化** — 标签管理中的手动去重循环改为 `dict.fromkeys()` 保持顺序去重
- **Emoji 映射统一** — 类型 emoji 映射从三处重复定义提取到 `models.TYPE_EMOJI` 共享常量

### 📊 测试
- 216 个测试全绿通过（新增 + 更新适配）
- 新增 `batch_set_favorite()` 方法覆盖测试（继承现有模式）
